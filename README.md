# רישום לידות 2026 - PWA

אפליקציית ווב פרוגרסיבית (PWA) לניהול רשומות לידות לשנת 2026.

## 📥 התקנת האפליקציה

### דרישות:
- דפדפן מודרני (Chrome, Edge, Firefox, Safari)
- שרת מקומי (לצורך התקנה)

### איך להריץ את האפליקציה:

#### אופציה 1: שרת Python (מומלץ)
```bash
python3 server.py
```
פתח את הדפדפן בכתובת: `http://localhost:8000`

#### אופציה 2: שרת Node.js
```bash
npx http-server -p 8000
```

#### אופציה 3: שרת PHP
```bash
php -S localhost:8000
```

### 📱 התקנת האפליקציה במכשיר:

1. **Chrome/Edge (Windows/Mac/Linux)**:
   - פתח את האפליקציה בכתובת `http://localhost:8000`
   - לחץ על כפתור "הורד אפליקציה" או
   - לחץ על סמל ההתקנה בשורת הכתובת

2. **Firefox**:
   - פתח את האפליקציה
   - לחץ על התפריט (☰) > "התקן"

3. **Safari (iOS)**:
   - פתח את האפליקציה ב-Safari
   - לחץ על כפתור Share (שיתוף)
   - בחר "הוסף למסך הבית"

4. **Chrome (Android)**:
   - פתח את האפליקציה
   - לחץ על התפריט (⋮)
   - בחר "הוסף למסך הבית"

## ✨ תכונות:

- ✅ הוספת רשומות חדשות
- ✅ עריכת ומחיקת רשומות
- ✅ חיפוש וסינון לפי חודש
- ✅ שמירת נתונים מקומית (localStorage)
- ✅ עבודה offline
- ✅ התקנה כאפליקציה עצמאית
- ✅ ממשק בעברית עם תמיכה ב-RTL

## 📂 קבצים:

- `index.html` - דף HTML ראשי
- `styles.css` - עיצוב
- `app.js` - לוגיקה של האפליקציה
- `manifest.json` - הגדרות PWA
- `sw.js` - Service Worker לעבודה offline
- `server.py` - שרת מקומי לבדיקות

## 🔧 פיתוח:

לפיתוח ובדיקות, הפעל שרת מקומי והשתמש בכלי פיתוח הדפדפן.

הנתונים נשמרים ב-localStorage של הדפדפן.

## 📝 הערות:

- האפליקציה חייבת לרוץ על HTTP/HTTPS כדי להיות ניתנת להתקנה
- שרת מקומי (`localhost`) מספיק לצורך התקנה
- הנתונים נשמרים במכשיר המקומי בלבד

## 🌐 פרסום באינטרנט:

כדי לפרסם את האפליקציה:
1. העלה את כל הקבצים לשרת אינטרנט (HTTPS)
2. ודא שהקובץ `manifest.json` נגיש
3. ודא שה-service worker נרשם בהצלחה
